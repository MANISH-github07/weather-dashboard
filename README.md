🌦️ Weather Dashboard

This is a weather dashboard that I built to improve my JavaScript skills and gain hands-on experience with REST APIs. The application allows users to search for any city and view its current weather along with a 5-day forecast.

The main goal of this project was to understand how to fetch data from an external API, work with asynchronous JavaScript, and display dynamic data on a responsive user interface.

Features :-
Search weather by city name
Display current temperature
Feels like temperature
Humidity
Wind speed
Weather condition
5-day weather forecast
Local time of the selected city
Recent searches stored using Local Storage
Responsive layout for different screen sizes

Built With :-
HTML5
CSS3
JavaScript (ES6)
Fetch API
OpenWeather API
Local Storage
